Zebra

This font is free to use, created by hand with the Fonty application.

Here you have the repo with the Zebra font, and the code to use it:


    @font-face {
        font-family: "Zebra";
        src: local("?"), url("Zebra.woff") format("woff"), url("Zebra.ttf") format("truetype");
    }
    .zebra { /* I recommend especially that it be used with a class */
        font-family: "Zebra";
        font-weight: 500;
    }
